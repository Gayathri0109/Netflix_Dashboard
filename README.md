## Netflix_Dashboard
This project focuses on analyzing Netflix movies and TV shows data to understand content trends and distribution across different categories. Using Microsoft Excel, the dataset was cleaned, organized, and analyzed to identify insights related to content type, genres, release years, ratings, and country-wise availability. An interactive dashboard was created using Pivot Tables, charts, and slicers to visualize key trends and patterns in Netflix’s content library.

## Objectives
Analyze the distribution of movies and TV shows on Netfli
Identify popular genres and content trends over the years
Understand country-wise content availability
Visualize ratings and release year patterns
Create an interactive dashboard for easy analysis

## Tools & Technologies Used
Microsoft Excel
Pivot Tables & Pivot Charts
Excel Functions (IF, COUNTIF, VLOOKUP/XLOOKUP, etc.)
Filters and Slicers
Data Cleaning & Formatting

## Key Insights
Movies make up a larger portion of Netflix’s content compared to TV shows
Certain genres like Drama and International Movies dominate the platform
Significant growth in content releases is observed after 2015
The United States and India contribute a major share of content
